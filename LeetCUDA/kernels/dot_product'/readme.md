# Dot Product

## 0x00 Explanation

Includes the following:

- [X] dot_product_f32_f32_kernel
- [X] dot_product_f32x4_f32_kernel (float4 vectorized version)
- [X] dot_product_f16_f32_kernel (FP16 version, using FP32 ACC)
- [X] dot_product_f16x2_f32_kernel (FP16 vectorized version, using FP32 ACC)
- [X] dot_product_f16x8_pack_f32_kernel (FP16 vectorized version, using FP32 ACC, Pack)
- [X] PyTorch bindings
