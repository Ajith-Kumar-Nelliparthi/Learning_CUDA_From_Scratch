# Sigmoid

## 0x00 Explanation

Includes the following:

- [X] sigmoid_f32_kernel
- [X] sigmoid_f32x4_kernel(float4 vectorized version)
- [X] sigmoid_f16_kernel
- [X] sigmoid_f16x2_kernel(half2 vectorization)
- [X] sigmoid_f16x8_kernel(unpack version)
- [X] sigmoid_f16x8_pack_kernel(pack version)
- [X] PyTorch bindings
