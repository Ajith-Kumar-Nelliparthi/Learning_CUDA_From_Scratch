# Relu

## 0x00 Explanation

Includes the following:

- [X] relu_f32_kernel
- [X] relu_f32x4_kernel(float4 vectorized version)
- [X] relu_f16_kernel
- [X] relu_f16x2_kernel(half2 vectorization)
- [X] relu_f16x8_kernel(unpack version)
- [X] relu_f16x8_pack_kernel(pack version)
- [X] PyTorch bindings
